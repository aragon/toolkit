# NormalizedHtml

Apply a default style to its content.

## Usage

```jsx
function App() {
  return (
    <NormalizedHtml>
      <h1>My title</h1>
    </NormalizedHtml>
  )
}
```

## Props

### `children`

| Type   | Default value   |
| ------ | --------------- |
| `Node` | None (required) |

The content that will receive the styling.
